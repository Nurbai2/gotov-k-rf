# 🇷🇺 Готов к РФ — Подготовка к жизни в России

Комплексное веб-приложение для адаптации и подготовки иностранных граждан к жизни в России. Включает интерактивные курсы русского языка, историю, законодательство, справочник городов и систему тестирования с выдачей сертификатов.

## ✨ Возможности
- 🔐 **Регистрация через Email** с подтверждением кода (SMTP)
- 📚 **Русский язык**: 15 грамматических тем, 800+ слов, разговорник
- 🏛️ **История**: таймлайн из 60+ событий от Рюрика до современности
- ⚖️ **Законы РФ**: миграция, трудовое право, налоги, Конституция
- 🏙️ **Города**: 25 городов с историей, менталитетом и достопримечательностями
- 🎓 **Тесты**: 5 категорий, автоматическая проверка и сохранение сертификатов
- 🌐 **Мультиязычность**: интерфейс на 16 языках (автоматический перевод + кэш)
- 📱 **Адаптивный дизайн**: корректно работает на ПК и мобильных устройствах

## 🛠️ Технологический стек
| Слой | Технологии |
|------|------------|
| **Frontend** | HTML5, CSS3 (CSS Variables, Grid/Flex), Vanilla JS (ES6+) |
| **Backend** | Node.js, Express, SQLite3, JWT, bcrypt |
| **Почта** | Nodemailer (SMTP) |
| **API** | MyMemory Translation API, RESTful endpoints |

## 📦 Установка и запуск

### 1. Требования
- [Node.js](https://nodejs.org/) версии 16+
- Любой современный браузер

### 2. Настройка Backend
```bash
# Перейдите в папку сервера
cd backend

# Установите зависимости
npm install

# Создайте файл .env (см. раздел ниже)
